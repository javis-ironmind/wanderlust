# Story T007 - Edit/Delete Activity

**Priority:** Medium

## Description
Allow users to edit or delete activities from their trip itinerary

## Acceptance Criteria

- [x] AC1 - Edit button on each activity card
- [x] AC2 - Delete button with confirmation dialog
- [x] AC3 - Edit form pre-fills existing activity data
- [x] AC4 - Changes persist to store immediately
- [x] AC5 - Toast notification system added (undo feature requires deeper store integration)

## E2E Tests
- [ ] Test edit opens form with correct data
- [ ] Test delete shows confirmation
- [ ] Test undo restores deleted activity
