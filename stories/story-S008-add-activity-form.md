# Story S008: Add Activity Form

## As a
User

## I want
A form to add new activities to a day

## So that
I can plan what's happening each day

## Acceptance Criteria
- [x] AC1 - Modal opens when "Add Activity" clicked
- [x] AC2 - Form fields: title (required), description, category (dropdown), start time, end time, location search, notes, cost
- [x] AC3 - Category dropdown with all categories and icons
- [x] AC4 - Time pickers for start/end
- [x] AC5 - Location search with autocomplete
- [x] AC6 - Form validates required title
- [x] AC7 - Save adds activity to day
- [x] AC8 - Cancel closes modal without saving
- [x] AC9 - Activity appears in correct chronological position
- [x] AC10 - Form resets on close
