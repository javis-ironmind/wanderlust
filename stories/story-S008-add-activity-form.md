# Story S008: Add Activity Form

## As a
User

## I want
A form to add new activities to a day

## So that
I can plan what's happening each day

## Acceptance Criteria
- [ ] AC1 - Modal opens when "Add Activity" clicked
- [ ] AC2 - Form fields: title (required), description, category (dropdown), start time, end time, location search, notes, cost
- [ ] AC3 - Category dropdown with all categories and icons
- [ ] AC4 - Time pickers for start/end
- [ ] AC5 - Location search with autocomplete
- [ ] AC6 - Form validates required title
- [ ] AC7 - Save adds activity to day
- [ ] AC8 - Cancel closes modal without saving
- [ ] AC9 - Activity appears in correct chronological position
- [ ] AC10 - Form resets on close
