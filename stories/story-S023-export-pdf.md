# Story S023: Export Trip as PDF

## User Story
As a traveler, I want to export my trip itinerary as a PDF so that I can have an offline copy for reference during travel.

## Acceptance Criteria
- [ ] Add "Export PDF" button to trip detail view
- [ ] Generate PDF with trip overview
- [ ] Include all activities with dates/times/locations
- [ ] Include flight and hotel bookings
- [ ] Apply clean, printable styling

## Technical Notes
- Use jsPDF or react-pdf library
- Generate on client side
- Include trip summary header
- List activities grouped by day
