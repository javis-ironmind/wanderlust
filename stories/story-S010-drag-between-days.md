# Story S010: Drag Activity to Different Day

## As a
User

## I want
To drag an activity from one day to another

## So that
I can move activities between days easily

## Acceptance Criteria
- [ ] AC1 - Can drag activity to different day container
- [ ] AC2 - Source day removes activity
- [ ] AC3 - Target day adds activity
- [ ] AC4 - Activity order preserved or added to end
- [ ] AC5 - Visual feedback shows which day is target
- [ ] AC6 - Confirmation not required for move
- [ ] AC7 - Move persists after completion
- [ ] AC8 - Can move between any days in trip
- [ ] AC9 - Day activity counts update correctly
- [ ] AC10 - Works alongside within-day drag
