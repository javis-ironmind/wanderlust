# Story S010: Drag Activity to Different Day

## As a
User

## I want
To drag an activity from one day to another

## So that
I can move activities between days easily

## Acceptance Criteria
- [x] AC1 - Can drag activity to different day container
- [x] AC2 - Source day removes activity
- [x] AC3 - Target day adds activity
- [x] AC4 - Activity order preserved or added to end
- [x] AC5 - Visual feedback shows which day is target
- [x] AC6 - Confirmation not required for move
- [x] AC7 - Move persists after completion
- [x] AC8 - Can move between any days in trip
- [x] AC9 - Day activity counts update correctly
- [x] AC10 - Works alongside within-day drag
