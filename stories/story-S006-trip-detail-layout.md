# Story S006: Trip Detail Page Layout

## As a
User

## I want
A well-designed trip detail page showing itinerary and map

## So that
I can view and edit my trip

## Acceptance Criteria
- [x] AC1 - Page displays at /trips/[tripId] route
- [x] AC2 - Header shows trip name, dates, cover image
- [x] AC3 - Two-column layout: itinerary (left), map (right)
- [x] AC4 - Day tabs or vertical list for navigation
- [x] AC5 - "Add Day" button visible
- [x] AC6 - "Add Activity" button per day
- [x] AC7 - Flight section visible
- [x] AC8 - Hotel section visible
- [x] AC9 - Responsive: stacks on tablet/mobile
- [x] AC10 - Loading skeleton while data loads
