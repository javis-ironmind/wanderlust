# Story S006: Trip Detail Page Layout

## As a
User

## I want
A well-designed trip detail page showing itinerary and map

## So that
I can view and edit my trip

## Acceptance Criteria
- [ ] AC1 - Page displays at /trips/[tripId] route
- [ ] AC2 - Header shows trip name, dates, cover image
- [ ] AC3 - Two-column layout: itinerary (left), map (right)
- [ ] AC4 - Day tabs or vertical list for navigation
- [ ] AC5 - "Add Day" button visible
- [ ] AC6 - "Add Activity" button per day
- [ ] AC7 - Flight section visible
- [ ] AC8 - Hotel section visible
- [ ] AC9 - Responsive: stacks on tablet/mobile
- [ ] AC10 - Loading skeleton while data loads
