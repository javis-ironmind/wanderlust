# Story S007: Activity Card Component

## As a
User

## I want
To see activity details in a card format

## So that
I can quickly understand what's planned

## Acceptance Criteria
- [ ] AC1 - Card shows: title, time, location, category icon, category badge
- [ ] AC2 - Category displayed with correct color
- [ ] AC3 - Time shown as "2:00 PM - 3:30 PM" or just start time
- [ ] AC4 - Location shown as name/address
- [ ] AC5 - Notes preview (truncated) if present
- [ ] AC6 - Edit button on hover
- [ ] AC7 - Delete button on hover (with confirmation)
- [ ] AC8 - Drag handle visible on left side
- [ ] AC9 - Hover effect on card
- [ ] AC10 - Empty time shows "Time TBD"
