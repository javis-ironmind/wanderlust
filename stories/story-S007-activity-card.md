# Story S007: Activity Card Component

## As a
User

## I want
To see activity details in a card format

## So that
I can quickly understand what's planned

## Acceptance Criteria
- [x] AC1 - Card shows: title, time, location, category icon, category badge
- [x] AC2 - Category displayed with correct color
- [x] AC3 - Time shown as "2:00 PM - 3:30 PM" or just start time
- [x] AC4 - Location shown as name/address
- [x] AC5 - Notes preview (truncated) if present
- [x] AC6 - Edit button on hover
- [x] AC7 - Delete button on hover (with confirmation)
- [x] AC8 - Drag handle visible on left side
- [x] AC9 - Hover effect on card
- [x] AC10 - Empty time shows "Time TBD"
