# Story S011: Flight Entry Form

## As a
User

## I want
To add flight details including airline, times, airports, and confirmation

## So that
I can keep all travel info in one place

## Acceptance Criteria
- [x] AC1 - "Add Flight" button in trip header
- [x] AC2 - Form fields: airline, flight number, departure airport + time, arrival airport + time, terminal, gate, confirmation, seat, notes
- [x] AC3 - Airline autocomplete from common airlines
- [x] AC4 - Airport search with IATA codes
- [x] AC5 - Date/time pickers for departure and arrival
- [x] AC6 - Required: airline, flight number, departure, arrival
- [x] AC7 - Save creates flight entry
- [x] AC8 - Flight appears in chronological order
- [x] AC9 - Can edit existing flight
- [x] AC10 - Can delete flight with confirmation
