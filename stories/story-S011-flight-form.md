# Story S011: Flight Entry Form

## As a
User

## I want
To add flight details including airline, times, airports, and confirmation

## So that
I can keep all travel info in one place

## Acceptance Criteria
- [ ] AC1 - "Add Flight" button in trip header
- [ ] AC2 - Form fields: airline, flight number, departure airport + time, arrival airport + time, terminal, gate, confirmation, seat, notes
- [ ] AC3 - Airline autocomplete from common airlines
- [ ] AC4 - Airport search with IATA codes
- [ ] AC5 - Date/time pickers for departure and arrival
- [ ] AC6 - Required: airline, flight number, departure, arrival
- [ ] AC7 - Save creates flight entry
- [ ] AC8 - Flight appears in chronological order
- [ ] AC9 - Can edit existing flight
- [ ] AC10 - Can delete flight with confirmation
