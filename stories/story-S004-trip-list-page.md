# Story S004: Trip List Page

## As a
User

## I want
To see all my trips on a main list page

## So that
I can select which trip to work on

## Acceptance Criteria
- [x] AC1 - Page displays at /trips route
- [x] AC2 - Grid of trip cards shown
- [x] AC3 - Each card shows: cover image, trip name, dates, day count
- [x] AC4 - Clicking card navigates to trip detail
- [x] AC5 - "Create Trip" button prominent
- [x] AC6 - Empty state shown when no trips
- [x] AC7 - Responsive grid (2 columns desktop, 1 mobile)
- [x] AC8 - Trip cards have hover effect
