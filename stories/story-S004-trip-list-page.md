# Story S004: Trip List Page

## As a
User

## I want
To see all my trips on a main list page

## So that
I can select which trip to work on

## Acceptance Criteria
- [ ] AC1 - Page displays at /trips route
- [ ] AC2 - Grid of trip cards shown
- [ ] AC3 - Each card shows: cover image, trip name, dates, day count
- [ ] AC4 - Clicking card navigates to trip detail
- [ ] AC5 - "Create Trip" button prominent
- [ ] AC6 - Empty state shown when no trips
- [ ] AC7 - Responsive grid (2 columns desktop, 1 mobile)
- [ ] AC8 - Trip cards have hover effect
