# Story S005: Create Trip Modal

## As a
User

## I want
A form to create a new trip with name, description, dates, and cover image

## So that
I can start planning a new trip

## Acceptance Criteria
- [ ] AC1 - Modal opens when "Create Trip" clicked
- [ ] AC2 - Form fields: name (required), description, start date, end date, cover image URL
- [ ] AC3 - Date picker for start/end dates
- [ ] AC4 - End date must be after start date (validation)
- [ ] AC5 - Cover image preview when URL entered
- [ ] AC6 - "Create" button disabled until required fields filled
- [ ] AC7 - Modal closes on successful create
- [ ] AC8 - New trip appears in list immediately
- [ ] AC9 - Form resets on close
- [ ] AC10 - Escape key closes modal
