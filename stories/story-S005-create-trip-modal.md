# Story S005: Create Trip Modal

## As a
User

## I want
A form to create a new trip with name, description, dates, and cover image

## So that
I can start planning a new trip

## Acceptance Criteria
- [x] AC1 - Modal opens when "Create Trip" clicked
- [x] AC2 - Form fields: name (required), description, start date, end date, cover image URL
- [x] AC3 - Date picker for start/end dates
- [x] AC4 - End date must be after start date (validation)
- [x] AC5 - Cover image preview when URL entered
- [x] AC6 - "Create" button disabled until required fields filled
- [x] AC7 - Modal closes on successful create
- [x] AC8 - New trip appears in list immediately
- [x] AC9 - Form resets on close
- [x] AC10 - Escape key closes modal
