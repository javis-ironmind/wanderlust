# Story S009: Drag to Reorder Within Day

## As a
User

## I want
To drag and reorder activities within a single day

## So that
I can easily change the sequence of activities

## Acceptance Criteria
- [x] AC1 - Drag handle visible on activity cards
- [x] AC2 - Clicking and dragging lifts the card with visual feedback
- [x] AC3 - Drop zones highlight when dragging over
- [x] AC4 - Release drops card in new position
- [x] AC5 - Order persists after drop
- [x] AC6 - Animation smooth on reorder
- [x] AC7 - Can drag from position 1 to position 5
- [x] AC8 - Other activities shift to accommodate
- [x] AC9 - No data loss on reorder
- [x] AC10 - Works on desktop browsers (Chrome, Firefox, Safari)
