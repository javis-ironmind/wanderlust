# Story S009: Drag to Reorder Within Day

## As a
User

## I want
To drag and reorder activities within a single day

## So that
I can easily change the sequence of activities

## Acceptance Criteria
- [ ] AC1 - Drag handle visible on activity cards
- [ ] AC2 - Clicking and dragging lifts the card with visual feedback
- [ ] AC3 - Drop zones highlight when dragging over
- [ ] AC4 - Release drops card in new position
- [ ] AC5 - Order persists after drop
- [ ] AC6 - Animation smooth on reorder
- [ ] AC7 - Can drag from position 1 to position 5
- [ ] AC8 - Other activities shift to accommodate
- [ ] AC9 - No data loss on reorder
- [ ] AC10 - Works on desktop browsers (Chrome, Firefox, Safari)
